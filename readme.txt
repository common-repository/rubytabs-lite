== Installation ==

**Automatic installation**

1. Go to the plugins menu, and choose “Add New”
1. Search for “rubytabs lite”, and then click “Install now”
1. Activate it



**Installing by FTP**

1. Download “rubytabs-lite.zip”
1. Extract the “rubytabs-lite.zip” directory to your computer
1. Upload the “rubytabs-lite.zip” directory to the “/wp-content/plugins/” directory
1. Activate the plugin in the Plugin dashboard


**Server Minimum Requirements**

* WordPress version 3.6
* PHP version 5.3.23
* MySQL version 5.0
